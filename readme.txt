Plugin for theme developers to preview and serve downloads of their themes.

== Description ==

Gives theme developers and their users a convenient dropdown menu to preview the themes currently installed on the site as well as providing a download link for the one currently selected.

There is an options page in the admin section that allows you to:

*	Choose which themes are available in the select menu
*	Change the label text for the select menu
*	Enable/disabled the [Easy Digital Downloads](https://wordpress.org/plugins/easy-digital-downloads/) support.
*	Enable/disabled the bar globally. If disabled globally, enable on pages/posts.
*	Enable and disable the default stylesheet
*	Add custom content to the bar

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Appearance->Theme Server screen to configure the plugin
